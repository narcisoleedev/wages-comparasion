# Calculadora de Redimentos 2026

Se você quer analisar quanto você vai receber, em um período de 12 meses, caso escolha uma certa vaga, essa calculadora faz as contas para você (por enquanto só funciona para CLT).

Basta colocar suas informações e um relatório com seus rendimentos será gerado, simples assim.

Pretendo adicionar mais features, mas fiz isso por enquanto só para analisas possibilidades de carreira :).

Para rodar, só instalar as dependências e rodar:

```
streamlit run main.py
```
